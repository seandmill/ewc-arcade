# EWC Arcade

**EWC Arcade** is a playful, interactive world-building and education platform designed for Ellis, Witten, and Calla. It combines creative construction with fun learning mini-games.

## Featured Games

### üèôÔ∏è City Builder

Create your own thriving metropolis!

- **Build & Expand:** Place roads, residential houses, shops, factories, and parks.
- **Manage Progress:** Watch your population grow and earn daily income from commercial and industrial zones.
- **Customization:** Design the layout of your city on a dynamic 30x30 grid.

### üßÆ Math Adventure

embark on a math journey with friendly companions!

- **Play with Buddies:** Choose from a variety of mascots like Penny Panda, Pete Penguin, or Max Monkey to help you solve problems.
- **Level Based Learning:** Progress through increasing difficulty levels covering addition, subtraction, and simple multiplication.
- **Earn Rewards:** Get stars for accuracy and coins to show off your math wiz status.

## Key Features

- **3D Avatar Customization:** Customize your player profile with a variety of 3D characters (Alex, Ben, Ava, Chloe, and more).
- **Interactive UI:** A vibrant, kid-friendly interface with smooth animations and sound effects.
- **Progress Tracking:** Earn achievements and track your progress across different games.

## Run Locally

**Prerequisites:** Node.js

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Launch:**
    ```bash
    npm run dev
    ```
